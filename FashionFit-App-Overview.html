01 App Overview

What Fashion Fit is, who it's for, and what problem it solves.

Fashion Fit is a mobile application that turns a user's physical wardrobe into a smart, AI-powered styling system. Users photograph their clothing, the AI analyses and categorises each item, and the app uses that data to generate personalised outfit recommendations based on occasion, weather, and the user's unique style identity (Style DNA). A fashion education coach (NOVA), a social feed, an outfit planner, wardrobe analytics, and a subscription system complete the platform.

🎯

Problem Solved

Most people own more clothes than they wear. Fashion Fit unlocks the full value of what someone already owns before suggesting anything new.

👤

Target User

Fashion-conscious individuals who want smarter, personalised outfit guidance without hiring a personal stylist.

🌍

Markets

Multilingual — English and Swahili supported natively. Designed for African and global markets.

📱

Platform

Android and iOS. Built with React Native + Expo for a single shared codebase across both platforms.

03 User Journey

The complete lifecycle of a Fashion Fit user from first open to daily active use.

1

Download & Open

User downloads the APK (Android) or installs via App Store (iOS). A branded loading screen plays on first open.

2

Register an Account

User creates an account with name, email, and password. A verification code is sent to their email. Access is blocked until the email is verified.

3

Onboarding & Style DNA Quiz

New users complete an onboarding quiz covering style preferences, lifestyle, occasions they dress for, and aesthetic goals. Answers seed the Style DNA profile that personalises all future AI interactions.

4

Grant Notification Permissions

The app requests push notification permissions. On grant, the Expo push token is saved to the backend and daily local outfit reminders (8am, 12:30pm) are scheduled on the device.

5

Build the Wardrobe

User photographs clothing items one at a time or in bulk (Pro+). Each image is uploaded to Cloudinary, then GPT-4o analyses the photo and returns category, colour palette, pattern, fabric type, occasion tags, and season suitability. All metadata is saved to the wardrobe database.

6

Request Outfit Recommendations

User selects an occasion and location. The app fetches live weather data, then the recommendation engine builds a prompt combining wardrobe data, Style DNA, weather, and occasion. GPT-4o-mini returns a complete outfit suggestion with items drawn from the user's actual wardrobe, scored by a rule-based fashion intelligence system.

7

Chat with NOVA

At any time, the user can open the chat and ask NOVA fashion questions — colour matching, what to wear to an event, how to style a specific item. NOVA replies in 3 sentences or fewer with emojis and one clear insight per message. Outfit suggestions are redirected to the Stylist feature.

8

Plan Outfits (Pro+)

User schedules outfits to specific calendar dates using the Outfit Planner — for events, trips, or the weekly rotation.

9

View Analytics (Pro+)

User reviews wardrobe composition charts, most-worn items, category gaps, and outfit frequency stats to make smarter wardrobe decisions.

10

Share on the Social Feed

User posts outfit photos to the community feed, follows other users, explores trending styles, and likes posts from people they follow.

11

Retention Loop

Daily notifications pull the user back. If inactive for 2 days, a re-engagement push is sent from the server. At 5 and 10 days, escalating messages are sent. The cycle keeps the user engaged long-term.

05 AI System

How artificial intelligence is used across every feature.

🔵

GPT-4o — Heavy Analysis

Used for tasks requiring deep visual and contextual understanding: clothing image analysis and Style DNA profile generation.

🟣

GPT-4o-mini — Fast Responses

Used for high-frequency, lower-latency tasks: outfit recommendations and NOVA chat replies.

Feature Model Input Output
Image Categorisation GPT-4o Clothing photo (base64 or Cloudinary URL) Category, colour palette, pattern, fabric, occasions, season, style tags
Style DNA Analysis GPT-4o Quiz answers + wardrobe composition summary Style identity profile, palette, aesthetic, occasion preferences, growth areas
Outfit Recommendations GPT-4o-mini Wardrobe items + Style DNA + weather + occasion Named outfit combination with items from the user's wardrobe + styling rationale
NOVA Chat GPT-4o-mini Chat history + wardrobe brief + Style DNA + user message Short, punchy fashion education reply (max 200 tokens, emoji-forward)
Fashion Intelligence Scoring Pure JS (no AI) Outfit combination + occasion + weather Numeric score for occasion fit, weather match, and style coherence

Style DNA — How it Works

Style DNA is the personalisation backbone of the entire app. It is built in two phases:

Phase 1 — Quiz
At onboarding, the user answers questions about their lifestyle, preferred occasions (work, casual, formal, night out), colour preferences, and style icons. These answers are sent to GPT-4o, which returns a structured Style DNA profile.
Phase 2 — Wardrobe Analysis
As the user uploads clothing items, the wardrobe composition (dominant colours, categories, occasion spread) is fed back into the Style DNA model to refine and evolve the profile over time.
Usage
Every outfit recommendation and NOVA chat session includes the user's Style DNA as context, ensuring all AI output is personalised to their aesthetic and not generic fashion advice.

07 AI Stylist (Outfit Recommendations)

How the recommendation engine builds outfits from the user's wardrobe.

1

User Inputs Occasion & Location

The user selects an occasion (casual, work, formal, night out, etc.) and optionally provides a location for weather data.

2

Weather Fetch

The backend fetches live weather conditions for the user's location. Destination weather (for travel planning) is available on Pro Yearly and Elite plans.

3

Wardrobe Context Assembly

The user's wardrobe items are loaded and described — type, colour, occasion tags, season. The Style DNA profile is included as context.

4

GPT-4o-mini Recommendation

The assembled context is sent to GPT-4o-mini, which selects items from the wardrobe and builds a complete outfit combination with a brief styling rationale.

5

Fashion Intelligence Scoring

The recommendation is scored by fashionIntelligence.js — a pure JavaScript rule engine that evaluates occasion fit, weather appropriateness, colour harmony, and style coherence. No additional AI cost.

6

Outfit Returned to User

The final outfit is displayed with item images from Cloudinary, the AI's styling notes, and an occasion-weather fit score.

Backend Deployment (Railway)

Push changes
Push to the connected GitHub branch. Railway auto-deploys on push.

All secrets (OPENAI_API_KEY, MONGODB_URI, CLOUDINARY keys, PAYSTACK keys, JWT_SECRET) are set in Railway's environment variable panel.
