AI Cost Strategy
gpt-4o is only used for image categorisation (vision required) and Style DNA initial analysis. Everything else uses gpt-4o-mini (17× cheaper). Style DNA has a 24h smart cache — no AI call made unless wardrobe changes by 3+ items. Outfit recommendations use a pure-JS algorithm first; mini only adds the final styling narrative. All calls are token-capped via max_tokens and logged to ApiUsage.
